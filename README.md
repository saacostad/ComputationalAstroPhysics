## Branch FundamentalAlgorithms

Ejercicios de las clases de algoritmos fundamentales

`Finite Differences/` contiene los ejercicios de clase y los 3 ejercicios de tarea sobre el tema

`Integration/` contiene los ejercicios de Riemman, Trapezio y Simpson.  

`Root Searching/` contiene el pdf de la tarea sobre bisection, Newton-Rhapson (single value y sistema 2x2) y secante, junto con el script usado para generar el $\LaTeX$.

`Regression/` contiene la tarea `Regression.py`, que consta de hacer la regresión de un set de datos exponenciales, primero, linealizándolos, y luego sin linealizar, utilizando el método de bisección.

## Branch DifferentialEquations

Contiene la primera tarea `Homework 1`, sobre implementar Euler en alguna ecuación física cualquiera. Igualmente, contiene la tarea `Runge-Kuta-exercise.py` sobre integrar una EDO (Circuito RC) y aplicar tanto regresión linearizada como método del secante. 

Las tareas de __EDOs de orden superior__ están en `SO_ODEs_homework.ipynb`.
